# Игровой ассистент (Game Deal & Info Bot)
[@GamerCompanionBot](https://web.telegram.org/k/#@GamerCompanionBot)

Telegram-бот для геймеров, предоставляющий:
- **Поиск игр**: Подробная информация через API RAWG.
- **Сравнение цен**: Поиск лучших предложений через API CheapShark.
- **Розыгрыши**: Отслеживание бесплатных игр с помощью парсинга FreeSteam.
- **Новости**: Последние игровые новости с помощью парсинга StopGame.
